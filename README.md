# Calculator_porject-
# 🧮 Beginner Python Calculator

A simple **command-line calculator** built in Python that performs basic arithmetic operations like addition, subtraction, multiplication, and division.

---

## 🚀 Features

* ➕ Addition
* ➖ Subtraction
* ✖️ Multiplication
* ➗ Division
* ⚠️ Handles division by zero
* 🔁 Continuous loop until valid operation

---

## 🛠️ Tech Stack

* Python 🐍
* CLI (Command Line Interface)

---

## 📊 Workflow (Graph)

```
        Start
          ↓
   Select Operation
          ↓
   Enter Numbers
          ↓
   Perform Calculation
          ↓
   Show Result
          ↓
        End
```

---

## 📌 How It Works

1. User selects an operation (1–4)
2. Inputs two numbers
3. Program performs calculation
4. Displays result
5. Handles invalid input or division errors

---

## 💻 Code Example

```python
def add(x, y):
    return x + y
```
